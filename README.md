# LMS Application

A Learning Management System (LMS) built with Next.js, TypeScript, and Ant Design.

---

## Features
- Login / Logout
- Protected routes
- Course CRUD (Create, Read, Update, Delete)
- Search and filter courses
- Responsive UI

---

## Tech Stack
- Next.js 14
- TypeScript
- Ant Design
- Axios
- Context API

---

## Installation

1. Clone the repository

git clone <repository-url>
cd lms-app

2. Install dependencies
npm install

3. Run the application
npm run dev

4. Open in browser
http://localhost:3000

## Login
- Email: any valid format (e.g. user@example.com)
- Password: at least 6 characters
Note: This project uses mock authentication.
